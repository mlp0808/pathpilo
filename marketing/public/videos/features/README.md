Drop your looping MP4 files in this folder using these names:

- `routeplanning-hero-loop.mp4`
- `routeplanning-map-loop.mp4`
- `routeplanning-mobile-loop.mp4`

Each video is used with:

- `autoplay`
- `muted`
- `loop`
- `playsinline`

If a video is missing, the page shows the matching poster placeholder image.
